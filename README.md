# Neovim configuration

Written exlusively in lua.

To use clone this repo into your `~/.config/` folder. Packer is used as the plugin manager so it needs to also be installed.

## Requirements
You will have to install packer for plugin manager:
```shell
yay -S nvim-packer-git
```
or
```shell
git clone --depth 1 https://github.com/wbthomason/packer.nvim\
 ~/.local/share/nvim/site/pack/packer/start/packer.nvim
```
